This link takes you to the home page of Apple. Which is a perfect example of a good wbsite layout.

  • C. Contrast: A white background with black contrasting text. Which is a great use of contrasting colors to make it easy for the user to read.
  • R. Repetition: repeated use of the same font, colors(black, white, and grey), and shapes(squares)
  • A. Alignment: All images and blocks of texts all align well together. They follow a simple alignment which seems to work effortlessly.
  • P. Proximity: All of the text is spaced out properly so it is easier for the user to read/navigate.
